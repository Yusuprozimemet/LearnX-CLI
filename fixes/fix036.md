# fix036 ŌĆö Three config-parsing edge cases crash instead of reporting a clear error

**Date:** 2026-05-16

## Symptom

Three distinct crashes with no user-friendly message:

1. A devloop.toml with an unbalanced quote in `e2e_tests` raises an unhandled `ValueError`
   from `shlex.split()` in `_build_e2e_command()`.
2. Passing a partial `config` dict (e.g., missing the `"review"` section) to
   `run_implement()` raises `KeyError` because `cfg["review"]["review_script"]` assumed
   both keys were present.
3. Calling `run_review.py --agents-dir` without a following value raises `IndexError`
   at `remaining[idx + 1]`.

## Root cause

1. `shlex.split()` raises `ValueError` on malformed shell syntax; the call was unguarded.
2. `cfg = config or _DEFAULTS` replaced the entire dict, so a partial caller-supplied
   config with only one section meant the other section's keys were absent.
3. The `--agents-dir` parser read `remaining[idx + 1]` without checking `idx + 1 < len(remaining)`.

## Fix

**`scripts/learnx_dk.py` ŌĆö `_build_e2e_command()`:**
```python
# before
inner = shlex.split(e2e_cmd)

# after
try:
    inner = shlex.split(e2e_cmd)
except ValueError as exc:
    print(f"error: invalid e2e_tests command in config: {exc}")
    sys.exit(1)
```

**`scripts/learnx_dk.py` ŌĆö `run_implement()`:**
```python
# before
cfg = config or _DEFAULTS
e2e_cmd = cfg["validation"]["e2e_tests"]
review_script = cfg["review"]["review_script"]

# after
cfg = config or {}
e2e_cmd = (
    cfg.get("validation", {}).get("e2e_tests")
    or _DEFAULTS["validation"]["e2e_tests"]
)
review_script = (
    cfg.get("review", {}).get("review_script")
    or _DEFAULTS["review"]["review_script"]
)
```

**`scripts/run_review.py` ŌĆö `main()`:**
```python
# before
agents_dir = remaining[idx + 1]

# after
if idx + 1 >= len(remaining):
    print("error: --agents-dir requires a value")
    sys.exit(1)
agents_dir = remaining[idx + 1]
```

## Tests added

- `test_build_e2e_command_exits_on_malformed_cmd` ŌĆö passes `"bad 'unbalanced"` and
  asserts `SystemExit(1)`.
- `test_run_implement_partial_config_falls_back_to_defaults` ŌĆö passes a config with
  only a `"validation"` section; asserts the default `run_review.py` path appears in
  dry-run output.
- `test_review_agents_dir_missing_value_exits` ŌĆö calls `main(["--agents-dir"])` and
  asserts `SystemExit(1)`.

## Rules for future work

1. Always guard `shlex.split()` calls on user-supplied strings ŌĆö malformed shell syntax
   raises `ValueError`, not `SyntaxError`.
2. Per-key fallback (`cfg.get(section, {}).get(key) or default`) is safer than
   whole-dict substitution (`cfg or defaults`) when callers may supply partial dicts.
3. Any CLI flag that consumes the next token must validate `idx + 1 < len(args)` before
   indexing, or use `argparse` to get bounds checking for free.
